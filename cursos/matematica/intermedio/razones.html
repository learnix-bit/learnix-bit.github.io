Razones y Proporciones — Nivel Avanzado

Profundización en relaciones, proporciones complejas y problemas avanzados. Incluye ejemplos paso a paso, ejercicios autocorregibles y mini quiz interactivo.

1. Definición Detallada

Una razón compara dos cantidades del mismo tipo mediante división: a:b = a/b. Una proporción establece que dos razones son equivalentes: a:b = c:d, donde a·d = b·c. En nivel avanzado, se aplican razones compuestas, proporciones múltiples y problemas de regla de tres combinados.

2. Propiedades y Fórmulas

  • Proporción básica: a/b = c/d → a·d = b·c
  • Razón compuesta: (a1 + a2 + …)/(b1 + b2 + …)
  • Regla de tres simple: a : b = c : x → x = (b·c)/a
  • Regla de tres compuesta: Multiplica o divide varias razones para encontrar la incógnita
  • Proporciones múltiples: a:b = c:d = e:f → se pueden usar productos cruzados múltiples

Tip memotécnico: "Producto cruzado nunca miente: multiplica extremos y medios".
