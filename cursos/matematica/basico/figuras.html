Figuras Geométricas — Profundización

Aprende las principales figuras geométricas, con fórmulas, propiedades, ejemplos, ejercicios interactivos y mini-quiz con retroalimentación.

Definición y Clasificación

Figuras planas: Triángulo, cuadrado, rectángulo, círculo. Solo tienen longitud y ancho.

Figuras sólidas: Cubo, prisma, esfera, cono. Tienen largo, ancho y alto (volumen).

Fórmulas importantes

  • Triángulo: Área = (base × altura)/2, Perímetro = suma de lados
  • Cuadrado: Área = lado², Perímetro = 4 × lado
  • Rectángulo: Área = base × altura, Perímetro = 2×(base+altura)
  • Círculo: Área = π×radio², Perímetro = 2×π×radio
  • Cubo: Volumen = lado³, Área superficial = 6×lado²
  • Prisma: Volumen = Área_base × altura
  • Esfera: Volumen = 4/3 × π × radio³, Área = 4×π×radio²
  • Cono: Volumen = 1/3 × π × radio² × altura

Ejemplos resueltos

Ejemplo 1: Área de un triángulo b=6, h=4 → A=(6×4)/2=12
Ejemplo 2: Volumen de un cubo lado=3 → V=3³=27
Ejemplo 3: Área de un círculo r=5 → A=π×5² ≈ 78.54
