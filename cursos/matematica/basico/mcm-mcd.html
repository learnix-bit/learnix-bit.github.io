MCM y MCD — Profundización

Definición detallada, propiedades, fórmulas, ejemplos, ejercicios autocorregibles y mini quiz interactivo con puntos globales.

1. Definición Detallada

Mínimo Común Múltiplo (MCM): Es el menor número positivo que es múltiplo de dos o más números.

Máximo Común Divisor (MCD): Es el mayor número que divide exactamente a dos o más números.

Ejemplo: para 12 y 18 → MCM=36, MCD=6

Tip memotécnico: "Mínimo arriba, Máximo abajo". Para MCM multiplicamos múltiplos, para MCD buscamos divisores.

2. Propiedades y Fórmulas

  • MCM(a,b) × MCD(a,b) = a × b
  • MCM(a,b,c) = MCM(MCM(a,b),c)
  • MCD(a,b,c) = MCD(MCD(a,b),c)
  • MCM siempre ≥ números dados
  • MCD siempre ≤ números dados
