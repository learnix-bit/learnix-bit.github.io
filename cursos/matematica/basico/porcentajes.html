Porcentajes y Proporciones — Profundización

Aprende todo sobre porcentajes y proporciones, con teoría detallada, fórmulas, ejemplos, ejercicios autocorregibles, mini-quiz interactivo y animaciones profesionales.

Definición

Porcentaje: Es la manera de expresar una cantidad como fracción de 100. Fórmula: P = (parte / total) × 100%.

Proporción: Es una igualdad entre dos razones o fracciones. Por ejemplo: a/b = c/d. Se cumple la propiedad del producto cruzado: a×d = b×c.

Propiedades y fórmulas

  • Para porcentajes: Aumento → Valor final = Valor inicial × (1 + p/100)
  • Para porcentajes: Descuento → Valor final = Valor inicial × (1 - p/100)
  • Proporciones: Producto cruzado → a/b = c/d → a×d = b×c
  • Proporciones directas: Si a/b = c/d, al aumentar a, aumenta c proporcionalmente
  • Proporciones inversas: Si a×b = c×d, al aumentar a, disminuye b para mantener constante

Ejemplos resueltos

Porcentaje: ¿Qué porcentaje es 25 de 200?

P = (25/200) × 100 = 12.5%

Proporción: Si 3 lápices cuestan 6 soles, ¿cuánto cuestan 5 lápices?

3/6 = 5/x → x = (5×6)/3 = 10 soles
